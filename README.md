# Smart Digital Lock System Using VHDL

## Description
This project implements a smart digital lock system using VHDL. The system provides secure password-based authentication for access control applications.

## Features
- Password verification
- Secure digital authentication
- Access control mechanism
- VHDL-based design and simulation

## Technologies Used
- VHDL
- FPGA/Digital Logic Design
- Simulation Tools
